# Real Estate Catalog (OOP in C++)

This is a prototype of an app that would allow users to manage a catalog of real estate assets.
It was built in C++ using OOP concepts.

## Features

* CRUD operations for three types of residences
* Filtering and sorting options for read operations
* Compact view showing only the type and price of each residence

## OOP Concepts Used

* Diamond inheritance
* Polymorphism
* Virtual functions
* Virtual inheritance
* Type casting
